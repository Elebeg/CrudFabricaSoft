# 🧙‍♂️ RPG Manager API

API construída com NestJS e TypeORM para gerenciamento de personagens e itens mágicos em um jogo de RPG.

---

## 🚀 Tecnologias

- NestJS
- TypeORM
- PostgreSQL
- Swagger (OpenAPI)
